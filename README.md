# 项目简介

该平台是一个基于Django框架和Python语言的B站弹幕数据爬取与分析工具，旨在为用户提供视频及弹幕信息分析、弹幕发送量分析、弹幕词云图分析、用户弹幕量分析和弹幕情感占比分析等功能。

## 核心功能
- 视频及弹幕信息分析
- 弹幕发送量分析
- 弹幕词云图分析
- 用户弹幕量分析
- 弹幕情感占比分析

## 适用场景
- 研究B站弹幕用户行为
- 分析弹幕内容的趋势与特点
- 为视频制作和内容运营提供数据支持

## 技术特点
- 采用Django框架，具有良好的安全性、可扩展性和易维护性
- 使用Pyecharts进行数据可视化，图表清晰直观
- 基于sqlite数据库，方便数据存储与管理
